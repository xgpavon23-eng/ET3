Documentación de la API - Sistema de Gestión de Entidades

Clase entidadGeneral

Clase principal para gestionar entidades dinámicamente.

constructor(nombreEntidad)

Descripción: Inicializa una nueva instancia para gestionar una entidad.

Parámetro Tipo Descripción
nombreEntidad String Nombre de la entidad a gestionar

Output: Instancia de entidadGeneral

createForm(accion, datos)

Descripción: Crea dinámicamente un formulario basado en la estructura de la entidad.

Parámetro Tipo Descripción
accion String Acción a realizar: 'ADD', 'EDIT', 'DELETE', 'SEARCH', 'SHOWCURRENT'
datos Object Datos a pre-cargar en el formulario (opcional)

Output: Void - Crea el formulario en el DOM

validarCampo(nombreAtributo, valor, accion)

Descripción: Valida un campo individual según las reglas definidas.

Parámetro Tipo Descripción
nombreAtributo String Nombre del atributo a validar
valor String/Number Valor del campo
accion String Acción en curso

Output: Boolean - true si es válido, false si no

Clase Dom

Clase para manipulación del DOM.

crearSeleccionablecolumnas(columnasSeleccionadas, todasColumnas)

Descripción: Crea un select múltiple para elegir columnas a mostrar.

Parámetro Tipo Descripción
columnasSeleccionadas Array Columnas actualmente seleccionadas
todasColumnas Array Todas las columnas disponibles

Output: Void - Actualiza el elemento select en el DOM

Clase Validations

Clase para validaciones generales.

validarFecha(fecha)

Descripción: Valida si una cadena es una fecha válida.

Parámetro Tipo Descripción
fecha String Fecha en formato YYYY-MM-DD

Output: Boolean - true si es fecha válida

validarEmail(email)

Descripción: Valida formato de email.

Parámetro Tipo Descripción
email String Dirección de email

Output: Boolean - true si es email válido
